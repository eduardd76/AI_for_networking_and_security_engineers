## üéØ Quick Start (3 Steps)

### Step 1: Click a Badge Above
Click any "Open in Colab" badge to open the notebook in your browser.

### Step 2: Get API Keys
You'll need API keys (free tier available):

**Anthropic API Key** (Required for most examples):
1. Go to https://console.anthropic.com/
2. Sign up for free account
3. Get your API key (starts with `sk-ant-api03-`)

**OpenAI API Key** (Required for RAG examples):
1. Go to https://platform.openai.com/
2. Sign up for account
3. Get your API key (starts with `sk-proj-`)

### Step 3: Run the Notebook
1. Click on first cell (Setup section)
2. Press `Shift + Enter` to run
3. Enter your API key when prompted
4. Continue running cells one by one

## üí° How to Use Colab

### Running Cells
- **Run one cell**: Click cell, press `Shift + Enter`
- **Run all cells**: Menu ‚Üí Runtime ‚Üí Run all
- **Stop execution**: Menu ‚Üí Runtime ‚Üí Interrupt execution

### Setting API Keys (2 Options)

**Option 1: Use Colab Secrets** (Recommended - More Secure)
```python
1. Click the key icon üîë on the left sidebar
2. Add secret: Name = "ANTHROPIC_API_KEY", Value = your-key
3. Enable "Notebook access"
4. Run the setup cell - it will use your secret automatically
```

**Option 2: Enter Directly** (Quick - Less Secure)
```python
# The setup cell will prompt you:
from getpass import getpass
api_key = getpass('Enter your API key: ')
```

### Modifying Examples
- All code cells are editable
- Try changing questions, configs, or prompts
- Click "Copy to Drive" to save your changes

### Saving Your Work
- File ‚Üí Save a copy in Drive
- Your changes are saved to your Google Drive
- Reopen anytime to continue

**"Module not found"**
```python

```

**"Invalid API key"**
```python
# Re-enter your key
import os
os.environ['ANTHROPIC_API_KEY'] = 'sk-ant-api03-your-key-here'
```

**"Rate limit exceeded"**
```python
# Free tier: 5 requests/minute
# Solution: Wait 60 seconds or upgrade to paid tier
import time
time.sleep(60)  # Wait 1 minute
```

**"Runtime disconnected"**
```python

```
